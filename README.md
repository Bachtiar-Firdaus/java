# 📘 Belajar Java dari Nol hingga Mahir

Repository ini berisi kumpulan kode, contoh, dan latihan terstruktur untuk menguasai ekosistem **Java**, mulai dari konsep dasar hingga fitur modern (Java 21+).

## 🎯 Daftar Modul & Materi
Berikut adalah daftar folder materi yang tersedia dalam repository ini (disusun berdasarkan kategori):

### 1. Java Foundations
* `belajar-java-dasar`: Dasar-dasar sintaksis dan logika Java.
* `belajar-java-oop`: Pemrograman Berorientasi Objek.
* `belajar-java-standard-classes`: Mengenal Class bawaan Java yang umum digunakan.
* `belajar-java-generic`: Implementasi pemrograman generik.
* `belajar-java-collection`: Framework koleksi data di Java.
* `belajar-java-sequenced-collection`: Fitur koleksi terbaru untuk urutan data yang pasti.

### 2. Advanced Language Features
* `belajar-java-lambda`: Functional programming dengan Lambda expressions.
* `belajar-java-stream`: Manipulasi data koleksi secara deklaratif.
* `belajar-java-reflection`: Introspeksi kode pada saat runtime.
* `belajar-java-record`: Data carrier class yang ringkas (Java 14+).
* `belajar-java-sealed-class`: Kontrol hierarki inheritance (Java 17+).
* `belajar-java-date-time`: Pengelolaan waktu dengan API modern.
* `belajar-java-i18n`: Internasionalisasi aplikasi (multi-bahasa).

### 3. Concurrency & Performance
* `belajar-java-thread`: Pemrograman multithreading konvensional.
* `belajar-java-virtual-thread`: Implementasi lightweight threads (Project Loom).
* `belajar-java-resilience4j`: Fault tolerance (Circuit Breaker, Rate Limiter, dll).

### 4. Data, File, & Database
* `belajar-java-io`: Input & Output stream untuk file.
* `belajar-java-csv`: Parsing dan penulisan data format CSV.
* `belajar-java-json`: Integrasi data format JSON.
* `belajar-java-database`: Koneksi database menggunakan JDBC.
* `belajar-java-persistence-api`: Pemahaman dasar Jakarta Persistence.
* `belajar-java-validation`: Implementasi validasi data (Bean Validation).

### 5. Tools & Ecosystem
* `belajar-java-maven`: Manajemen dependensi dan build lifecycle.
* `belajar-java-unit-test`: Pengujian kode otomatis dengan JUnit.
* `belajar-java-logging`: Implementasi log aplikasi yang rapi.
* `belajar-java-lombok`: Mengurangi kode boilerplate (Getter, Setter, Constructor).
* `belajar-java-web-servlet`: Dasar pengembangan aplikasi web Java.

### 6. Projek & Studi Kasus
* `aplikasi-todo-list`: Implementasi dasar logika aplikasi.
* `studi-kasus-java-database-aplikasi-todolist`: Integrasi aplikasi Todo List dengan database nyata.

## 🚀 Persyaratan Sistem
Karena beberapa modul menggunakan fitur terbaru, sangat disarankan menggunakan:
- **JDK 21** atau versi yang lebih tinggi (LTS).
- **Maven** untuk manajemen dependensi.
